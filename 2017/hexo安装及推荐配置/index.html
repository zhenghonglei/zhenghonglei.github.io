hexo安装及推荐配置

发表于 2017-02-04 | 阅读次数

hexo官方网址 https://hexo.io/zh-cn/

hexo官方文档 https://hexo.io/docs/

hexo官方中文文档 https://hexo.io/zh-cn/docs/

本文提及的安装和配置均来源于官方文档，只是对官方文档加以提炼。如有不解，强烈推荐阅读官方文档

安装

前提

本机安装好node环境 可参考 mac下使用n搭建Node环境

本机安装好git客户端

开始安装

使用npm安装hexo提供的脚手架

npm install -g hexo-cli

创建博客

使用上面安装的hexo脚手架，创建也很简单

文件夹目录结构&含义请参考 hexo官方建站说明

配置博客

站点配置

站点配置文件，在<博客存放的文件夹名称>下的 _config.yml 文件中。

yml文件采用缩进代表层级关系，详细语法请参考阮一峰博客YAML语法

主题配置

推荐使用next，这个主题比较简洁，最主要是使用的人最多，跟大多数人的博客保持风格一致，可以降低别人阅读你博客时的部分困扰。
